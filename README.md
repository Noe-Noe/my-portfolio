# my-portfolio

A modern, glass-inspired personal portfolio website built with React and Tailwind CSS.  
This site showcases selected projects, including full-stack web applications, data-driven systems, and interactive 3D experiences.

## Overview

This portfolio is designed to present projects in a clear, structured way while keeping the interface minimal and focused on content.  
Each project includes a dedicated detail page with descriptions, features, technology stack, and media previews.

The site is fully responsive and optimized for performance and readability.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Three.js (for selected projects)
- Vercel (deployment)

## Featured Projects

- **Job Application Tracker**  
  A full-stack web application for managing job applications, statuses, and follow-ups, with plans for browser extension automation.

- **Interactive Website for Real Estate Price Prediction (Singapore)**  
  Final Year Project involving a full-stack architecture for predicting property prices using historical data, geospatial features, and interactive visualizations.

- **Animal 3D Interactive Website**  
  A Three.js-based project exploring interactive 3D scenes, camera controls, sound integration, and performance optimization.

- **International School Inquiry Website**  
  A project-focused informational website designed to present school programs and collect inquiries through a structured form flow.

## Key Features

- Multi-page layout with dedicated project detail views
- Clean, glass-style UI design
- Interactive project media (images and video)
- Click-to-zoom media modal
- Responsive layout for different screen sizes
- Modular data-driven project configuration
